# LLM-Delivery

## Overview
![DeliverBench](https://github.com/user-attachments/assets/b4776f9f-852c-4532-a5e6-dd5d8849aa2d)
LLM-Delivery is a benchmark implemented based on SimWorld siulator.

## Configuration
- Low level navigation task:
    - Consider the image observation
    - Output the low level function calling

- High level planning task:
    - No considering the image observation
    - No output low level function calling
    - Only input the description of the current state in text
    - Only output the description of the high level activity
